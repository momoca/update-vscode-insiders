# update-vscode-insiders
通过 PowerShell 静默更新 VS Code Insiders 便携版（Portable）版本的自动更新脚本。

## ✨ 背景与动机
VS Code Insiders 的便携版没有内置自动更新功能。  
手动更新时通常需要：

- 下载最新 ZIP 压缩包  
- 解压到临时目录  
- 覆盖旧文件  
- 删除旧版本目录  
- 保留 `data` 用户数据目录  

整个流程繁琐且容易出错。

**本脚本自动完成上述所有步骤，只需一条命令即可完成更新。**

---

## 🚀 功能特点
- 自动检测最新 VS Code Insiders 版本（基于官方重定向 URL）
- 自动下载 ZIP 包（支持断点续传、失败重试、BITS 下载）
- 自动校验文件大小，避免下载损坏
- 自动解压并识别新版本目录
- 自动覆盖更新（保留 `data`）
- 自动删除旧版本目录
- 自动清理临时文件
- 全程静默、无交互、无窗口弹出
- 兼容 Windows PowerShell 5.1

---

## 📦 使用方法

### 1. 将脚本放入 VS Code Insiders 便携版目录
例如：
```
D:\VSCode
├─ data
├─ 1234567890\   ← 当前版本目录
├─ update.ps1    ← 本脚本
```

### 2. 创建一个快捷方式用于一键更新
右键 → 新建快捷方式，目标填写：
`powershell.exe -ExecutionPolicy Bypass -File "D:\VSCode\update.ps1"`

之后只需双击快捷方式即可自动更新。

---

## 🛠 工作流程（脚本内部逻辑）
脚本执行时会自动完成以下步骤：

1. **检查远程版本号**  
   通过官方更新 URL 获取最新版本的 SHA。

2. **判断本地是否已是最新版本**  
   若本地已有对应版本目录，则直接退出。

3. **下载 ZIP 包**  
   - 支持 BITS 下载  
   - 支持断点续传（PowerShell 6+）  
   - 支持失败重试  
   - 校验文件大小一致性  

4. **解压到临时目录**  
   自动识别新版本目录名（10 位 SHA 前缀）。

5. **覆盖更新（保留 data）**  
   使用 `Copy-Item` 覆盖所有文件，但排除 `data`。

6. **删除旧版本目录**  
   自动识别并删除旧版本的 10 位 SHA 目录。

7. **清理临时文件**  
   删除 ZIP、临时目录、下载中间文件。

8. **显示更新完成信息并等待按键退出**
